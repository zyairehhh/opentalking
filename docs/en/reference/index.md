# Reference Materials

Reference materials contain a small set of pages that users may need to revisit, compare,
or audit. For now, this section only keeps Benchmark and Changelog so API, configuration,
and asset-format notes do not make the main documentation feel crowded while they are still
changing.

## What Is in This Section

### Benchmark

Records tested models, hardware, runtime backends, and key metrics such as FPS, first-frame
latency, startup time, end-to-end latency, and resource usage.

### Changelog

Tracks documentation and project capability changes, especially changes that affect upgrades,
deployment, or benchmark reproducibility.

## Later Candidates

API, configuration, and asset-format references can return here after they become stable.
Before that, they should live near the corresponding usage or model-support pages.
